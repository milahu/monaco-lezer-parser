# monaco-lezer-parser

[lezer-parser](https://github.com/lezer-parser/lr) syntax highlighting for [monaco-editor](https://github.com/microsoft/monaco-editor)

## Install

```bash
npm install -D https://github.com/milahu/monaco-lezer-parser
```
